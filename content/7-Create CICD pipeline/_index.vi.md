#### Tổng Quan
Để đơn giản hóa việc triển khai API web, chúng ta sẽ thiết lập một pipeline CI/CD với AWS CodePipeline. Điều này tự động hóa quy trình xây dựng, kiểm tra và triển khai, tích hợp với CodeBuild và Elastic Beanstalk. Tự động hóa nâng cao hiệu quả, giảm lỗi và tăng tốc độ giao hàng tính năng.
